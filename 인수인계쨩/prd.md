# 인수인계쨩 — PRD

## 목적
성과관리 시스템 인수인계 내용을 단일 HTML 파일로 구현한 대화형 가이드. 전사 DW와 7대 수기 관리 테이블을 결합해 고객별 수익성을 산출하는 핵심 로직, 예외 처리 규칙, 계산 예시를 시각화한다.

## 현행 파일
`index5.html` — 최신본 (1,500줄+)

## 섹션 구성

| # | 앵커 | 내용 |
|---|------|------|
| 1 | `#s1` | 시스템 구성 및 데이터 파이프라인 (파이프라인 다이어그램 + 7대 수기 테이블 + Join 구조) |
| 2 | `#s2` | 성과지표 산출 로직 (확정매출 보정 + 수익성 공식) |
| 3 | `#s3` | 특정 고객사 예외 처리 규칙 (네이버 코람코 IDC, 한전 KEPCO) |
| 4 | `#s4` | 산출 상세 예시 — 그룹A 시뮬레이션 2026.05 |
| 5 | `#s5` | 데이터 배치 및 릴리즈 일정 |

## 핵심 도메인 규칙

- **VRB 가중평균**: 반영이익 합계 ÷ 반영매출 합계 (단순평균 ❌)
- **반영개월수 기준**: '26년 1~5월 중 심의 유효 개월
  - 26년 내 종료: 1월~종료월 / 26년 이후 종료: 1~5월 전체 / 26년 2월 이후 시작: 시작월~5월 / 일회성: 해당 월 1회
- **예외 고객**: 네이버 코람코 IDC (Billing 역산), 한전 KEPCO 전력통신 (PL 이익률 강제 일괄)
- **수기 보정 필수**: 에스원(M21/M2M), KGM(T35/커넥티드카), 토요타(T35/커넥티드카) — 월평균 4~5억 원
- **배치 기준일**: 매월 11일 05:00~, 공휴일 무관
