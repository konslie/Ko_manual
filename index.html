조정 배경: MVNO 정산 매출은 사업팀으로 귀속된 뒤, UCube 매출보정 기능으로 영업팀으로 귀속처리함. 단, 매출보정처리 시 사업자번호 없이 조정처리하여 고객단위 매출에서 제외되는 문제 존재. 이를 보완하고자 수기 조정 처리 필수. 조정방법: 기업DW 확정매출 업데이트 시 각 영업사원의 손익별 '보정매출' 금액 확인 (또는 파트너플러스에서 각 사업자별 전표금액 확인)

2 손익별 가중평균 이익률 산출

동일 손익에 여러 심의가 존재 시, 반영이익 합계 ÷ 반영매출 합계로 가중평균 처리

'인터넷'은 규모가 큰 일회성 심의로 인해 이익률 하락
📞 전화 (A)
반영매출: 9,000만 + 2억 = 2억 9,000만
반영이익: 1,080만 + 2,800만 = 3,880만
3,880만 / 2.9억 = 13.4%
🏢 IDC (I)
반영매출: 3억 + 9,000만 = 3억 9,000만
반영이익: 2,400만 + 900만 = 3,300만
3,300만 / 3.9억 = 8.5%
🌐 인터넷 (B)
반영매출: 1억 + 2,000만 + 10억 = 11억 2,000만
반영이익: 1,500만 + 260만 + 3,200만 = 4,960만
4,960만 / 11.2억 = 4.4%
